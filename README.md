# 📝 Django Blog

A blog web application built using **Django** while learning backend web development. This project follows best practices such as template inheritance, reusable layouts, Bootstrap integration, and Django's URL routing system.

> 🚧 This project is actively being developed, and new features will be added over time.

---

## 🚀 Features

- ✅ Django template inheritance
- ✅ Responsive Bootstrap 5 interface
- ✅ Navigation bar
- ✅ Home and About pages
- ✅ Static files integration
- ✅ Dynamic blog posts using Django templates

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

Move into the project directory:

```bash
cd <repository-name>
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install the dependencies:

```bash
pip install django
```

Run the development server:

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
